# const_ohmyzsh
My custom theme for Oh-my-zsh

## Features

* User name
* Machine name
* Current time
* Full path to current working directory
* Current working directory
* Git branch
* Git status
    * `>>` —— Dirty branch
    * `✚` —— Added files
    * `✖` —— Deleted files
    * `*` —— Modified files
    * `»` —— Renamed files
    * `=` —— Unmerged changes
    * `?` —— Untracked changes
* Prompt indicator and full path turns red if the last run fails
